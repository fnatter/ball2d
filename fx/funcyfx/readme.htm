If you are running Windows, please start this program only in fullscreen-mode (Alt+Enter at the DOS-Prompt).

Introduction

Use the tab-key to enter console-mode. This is where you specify functions and work with them. Press the tab-key again to return to full-screen-mode. Here you can "walk-around" in the x-y-plane, and scale it's size using X/Y and +/-. You have 4 functions you can works with (f/g/h/i). Use f/g/h/i = to assign functions. To show/hide a function or one of its derivatives, use show/hide. These and more commands are described below. You can choose any resolution that your VBE 2.0 supports by changing fx.cfg. Type ? for a list of commands at runtime.

Common commands

  • use f/g/h/i = to assign a function
    • the function will initially be shown, the two derivatives will be hidden (use show(f/g/h/i')
    • functions available for function-definitions are [arc]sin/cos/tan/cot; ln/log(x,y)/sqrt/abs/sgn
    • i.e.: f=(x^3 - 2*x^2 + 6*x)/abs(sin(x)+log(2,x))
  • use show/hide(f/g/h/i['/'']) to show or hide a function or one of its derivatives (1st and 2nd). i.e: show(f')
  • use intersect(f/g/h/i, f/g/h/i) to find the intersections between two functions (asaiw). i.e.: intersect(f,g)
  • use texit(, ) to export a function's representation to LaTeX2e
  • use integ(f/g/h/i, , ) to find a definite integral. i.e.: integ(f,0,2)
  • use range(, , , ) to specify a range. i.e.: range(-2*pi, 2*pi, -2, 2)
  • use eval() to evaluate an expression. i.e: eval(sin(3/4*pi))
  • use analyze(f/g/h/i) to analyze a function (asaiw). i.e.: analyze(g)
  • use redraw to redraw all visible functions (and derivatives)
  • use clear to hide all functions
  • use whatsup to see what each of the functions is set to.
