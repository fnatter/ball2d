If you are running Windows, please start this program only in fullscreen-mode (Alt+Enter at the DOS-Prompt).
  • cwsdpmi is the DPMI-server, you need it to run ball
  • you need a VBE 2.0 in order for this to run
  • just start ball with no parameters for a list of parameters
  • you can use CTRL-S and CTRL-L to zoom, but the background image will stay like it is
  • don't use more balls than fit on the screen (especially in onedee-mode) - if it halts, use Ctrl-Break to quit
  • if you want to look at the theory, start something like ball 10 -details -prompt (and maybe -onedee) - the radius is proportional to the mass, and the length of the little line that tells you the direction is proportional to the velocity
