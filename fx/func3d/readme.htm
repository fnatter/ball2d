If you are running Windows, please start this program only in fullscreen-mode (Alt+Enter at the DOS-Prompt).

Introduction

Func3D works by approximating a 3d-function by a number of polygons (specified by the "steps"-value) which you can then look@ from an arbitrary viewpoint. You can choose any resolution that your VBE 2.0 supports in the fx3d.cfg-file. When you start func3d.exe you'll be in the fullscreen-mode. Here you can "walk-around" in the "world" of the function with the movement similar to Descent:
Use a to accelerate forward, y to accelerate backward, UP/DOWN Arrow to pitch up/down and LEFT/RIGHT Arrow to turn left/right. Use the tab-key to enter console-mode. Here you can set a new function or change appearance values and the coordinates of the drawing plane. Type ? for a list of commands at runtime.

Common commands

  • use f = to assign a function
  • use fov = , where FOV is the new Field of View angle (0.0 <= FOV <= 180.0)
  • use type = , where facetype is one of the following: wireframe/flat/gouraud/textured/gouraud+textured
  • use steps = to change the number of steps when scanning. the number of polygons will be nSteps^2, and the number of Vertices will be (nSteps + 1)^2
  • use range(, , , ) to specify the range of the base plane. i.e.: range(-5, 5, -5, 5)
  • use eval() to evaluate an expression. i.e: eval(sin(3/4*pi))
