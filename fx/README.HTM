Installation

Hi, just type 'doconfig' and then 'make' in this directory to make release-versions of all projects ('doconfig' compiles a utility needed to compile, look at fx/util/chdep.c for details).
If you want to control compilation-options or compile only a few projects, look into the makefiles in the subdirectories.

Usage

If you are running Windows, please start these programs only in fullscreen-mode (Alt+Enter at the DOS-Prompt).

These programs are very cryptic and unintuitive to use. So please read subdir/readme.htm (except for util and incl, you might guess what these are for). You can exit every program with ESC.
